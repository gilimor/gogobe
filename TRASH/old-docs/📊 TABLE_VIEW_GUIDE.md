# 📊 מדריך תצוגת טבלה

## 🎯 מה זה?

**תצוגת טבלה** - אופציה חדשה לצפייה במוצרים בפורמט טבלה מסודר!

```yaml
יתרונות תצוגת טבלה:
  ✅ צפופה יותר - יותר מוצרים במסך
  ✅ קלה להשוואה - כל המחירים בשורה
  ✅ מיון קל - לחץ על כותרת
  ✅ סריקה מהירה - ראה הכל במבט אחד
  ✅ מקצועית - נראית כמו Excel

יתרונות תצוגת כרטיסים:
  ✅ ויזואלית - יותר יפה
  ✅ קלה לקריאה - מידע מאורגן
  ✅ Hover effects - אינטראקטיבית
  ✅ Mobile friendly - מתאים למובייל
```

---

## 🔄 איך לעבור בין תצוגות?

### בראש התוצאות תראה 2 כפתורים:

```
┌─────────────────────────────────┐
│ 805 תוצאות    [🔲] [📊]         │
└─────────────────────────────────┘
```

**🔲 = תצוגת כרטיסים (Grid)**
**📊 = תצוגת טבלה (Table)**

לחץ על הכפתור כדי לעבור!

---

## 📊 מה יש בטבלה?

### עמודות:

```yaml
1. שם המוצר:
   - שם מלא
   - עם Tooltip על hover
   
2. קטגוריה:
   - Badge צבעוני
   - קל לזיהוי

3. מחיר מינימלי:
   - ירוק
   - הזול ביותר

4. מחיר מקסימלי:
   - אפור
   - היקר ביותר

5. ממוצע:
   - כחול
   - ממוצע כל הספקים

6. ספקים:
   - 📊 מספר ספקים
   - כמה אופציות יש

7. עודכן:
   - תאריך עדכון אחרון
   - מתי נסרק

8. פעולות:
   - כפתור "פרטים"
   - פתיחת מודל
```

```yaml
  - רקע לבן

Colors:
```

## 💡 מתי להשתמש בכל תצוגה?

### 🔲 תצוגת כרטיסים (Grid):

```yaml
מתאים ל:
  ✅ גלישה כללית
  ✅ צפייה ראשונית
  ✅ כשרוצה משהו יפה
  ✅ במובייל
  ✅ כשיש תמונות (בעתיד)

  ❌ השוואת מחירים מהירה
  ❌ צפייה בהרבה מוצרים
```

## 🔧 טיפים טכניים

### CSS Classes:

```css
.products-table - הטבלה עצמה
.table-product-name - שם מוצר
.table-category - badge קטגוריה
.table-price-min - מחיר מינימלי (ירוק)
.table-price-max - מחיר מקסימלי (אפור)
.table-price-avg - ממוצע (כחול)
.table-suppliers - מספר ספקים
.table-date - תאריך
.table-btn - כפתור פעולה
```

---

### JavaScript:

```javascript
// עבור לתצוגת טבלה
switchView('table');

// עבור לתצוגת כרטיסים
switchView('grid');

// בדוק תצוגה נוכחית
console.log(currentView); // 'grid' or 'table'
```
