# ⚡ מדריך עיבוד מקבילי - KingStore

## 🎯 מה זה עושה?

במקום לחכות שהקובץ ירד ואז לעבד אותו, המערכת עושה **שניהם במקביל**:
- 🔽 **חלון 1**: מוריד קבצים ללא הפסקה
- ⚙️ **חלון 2**: מעבד קבצים ברגע שהם מגיעים

**תוצאה**: **פי 2-3 יותר מהר**! ⚡

---

## 🚀 איך להשתמש?

### אופציה 1: פשוט והכי מהיר (50 קבצים)
```bash
⚡ PARALLEL_50_FILES.bat
```

**מה קורה:**
1. נפתח חלון רקע שמעבד קבצים
2. מתחיל להוריד 50 קבצים
3. כל קובץ שמסיים - מיד מעובד!

⏱️ **זמן**: ~10-15 דקות במקום 30!

---

### אופציה 2: הכל! (1000 קבצים)
```bash
⚡ PARALLEL_SIMPLE.bat
```

זה יוריד את **כל הקבצים** הזמינים ויעבד אותם במקביל.

⚠️ **אזהרה**: יכול לקחת שעות!

---

## 📊 איך לבדוק שזה עובד?

### בזמן ההרצה:
- **חלון 1** (הורדות): תראה "Downloaded: Price123.gz"
- **חלון 2** (עיבוד): תראה "Processing: Price123.gz"

### אחרי:
```bash
📊 SHOW_KINGSTORE_INFO.bat
```

תראה:
- כמה קבצים הורדו
- כמה מוצרים יובאו
- כמה מחירים נוספו

### רק לעבד (ללא הורדות):
```bash
⚙️ PROCESS_DOWNLOADED_FILES.bat
```

### רק להוריד (ללא עיבוד):


## 💡 טיפים

### 1. השאר את חלון העיבוד פתוח
אם תסגור אותו בטעות - פתח שוב:
```bash
backend\database\parallel_processor_loop.bat
```

### 2. בדוק התקדמות
החלון מציג כל 5 שניות אם יש קבצים לעיבוד.

### 3. הרץ בזמן שאתה עושה דברים אחרים
זה רץ ברקע - אתה יכול לעבוד על דברים אחרים!
