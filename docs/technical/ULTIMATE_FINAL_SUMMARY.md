   ```bash
   scripts\setup\setup-environment.bat
   ```

   ```bash
   scripts\database\show-info.bat
   scripts\web\start-web.bat
   ```

3. ⏳ **מחק TRASH** (אחרי וידוא)
   ```bash
   rmdir /s /q TRASH
   ```
