## 🖥️ מה תראה באתר?

### 1. **Header (למעלה)**
```
🔍 Gogobe
השוואת מחירים חכמה

[14,527 מוצרים] [21 ספקים] [13,458 מחירים]
```

### 2. **Search Bar (חיפוש)**
```
[____חפש מוצר____] [🔍 חפש]
```

נסה לחפש:
- **חלב**
- **milk**  
- **לחם**
- **מים**

### 3. **Filters (סינונים)**
```
קטגוריה: [הכל ▼]
ספק: [הכל ▼]
מחיר מינימלי: [0]
מחיר מקסימלי: [1000]
מטבע: [ILS (₪) ▼]
מיון: [מחיר: נמוך לגבוה ▼]
```

### 4. **תוצאות**
טבלה עם:
- שם המוצר
- ברקוד
- יצרן/מותג
- קטגוריה
- חנויות
- מחירים (מינ, מקס, ממוצע)
- תאריך עדכון

### 5. **Footer (למטה)**
```
API: ✅ Connected
📊 ניהול מקורות | 🔍 Error Monitor
```

## 🎯 דוגמאות שימוש:

### חיפוש בסיסי:
1. הקלד "milk" בתיבת החיפוש
2. לחץ 🔍
3. תראה 3 מוצרים של NOT MILK

### חיפוש עם סינון מחיר:
1. חפש "milk"
2. הגדר מחיר מקסימלי: 20
3. תראה רק מוצרים עד 20 ₪

### חיפוש לפי קטגוריה:
1. בחר קטגוריה: "Dairy"
2. תראה את כל מוצרי החלב

## ⚠️ אם האתר לא עובד:

### בעיה: "לא ניתן להתחבר"
**פתרון:**
```bash
# בדוק שהשרת רץ
docker-compose ps

# אם לא רץ:
docker-compose up -d

# המתן 10 שניות
# נסה שוב
```

### בעיה: "שגיאה בטעינת נתונים"
**פתרון:**
```bash
# בדוק לוגים
docker-compose logs -f api

# אם יש שגיאת DB:
# בדוק שPostgreSQL רץ
```

### בעיה: "העיצוב לא נטען"
**פתרון:**
```bash
# רענן את הדפדפן
Ctrl+F5 (Windows)
Cmd+Shift+R (Mac)
```
