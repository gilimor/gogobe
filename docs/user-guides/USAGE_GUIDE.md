פתח בדפדפן: **http://localhost:8000**

## 🔍 חיפוש מוצרים

### חיפוש חופשי:
```
חלב → מציג את כל מוצרי החלב
שוקולד → מציג כל השוקולדים
קינג סטור → מציג מוצרים מהרשת
```

### סינון לפי קטגוריה:
1. פתח את התפריט "קטגוריה"
2. בחר קטגוריה (למשל: "חלב ומוצרי חלב")
3. לחץ "חפש"

### סינון לפי מחיר:
```
מחיר מינימלי: 5 ₪
מחיר מקסימלי: 20 ₪
```

### מיון:
- **מחיר: נמוך לגבוה** - הזול ביותר ראשון
- **מחיר: גבוה לנמוך** - היקר ביותר ראשון
- **שם: א-ת** - מיון אלפביתי
- **החדשים ביותר** - מוצרים שעודכנו לאחרונה

---

## 📊 הבנת הטבלה

### עמודות:

| עמודה | מה זה אומר |
|-------|------------|
| **שם המוצר** | שם המוצר בעברית |
| **ברקוד** | מזהה ייחודי (EAN/UPC) |
| **יצרן/מותג** | שם היצרן או קוד היצרן |
| **קטגוריה** | הקטגוריה הראשית |
| **חנויות** | שם הסניף (קינג סטור - סניף X) |
| **מחיר מינימלי** | המחיר הזול ביותר |
| **מחיר מקסימלי** | המחיר היקר ביותר |
| **ממוצע** | ממוצע כל המחירים |
| **עודכן** | מתי המחיר עודכן לאחרונה |

### כפתור "פרטים":
לחיצה על "פרטים" מציגה:
- **כל המחירים** מכל הסניפים
- **פרטי כמות** (משקל, יחידת מידה)
- **פרטי חנות** (שם, קוד)
- **תאריכי עדכון** של כל מחיר

## 💡 טיפים לחיפוש טוב

### 1. השתמש במילים חלקיות:
```
"חלב" → ימצא: חלב טרי, חלב מופחת, חלבה
"גבינ" → ימצא: גבינה, גבינת, גבינות
```

### 2. שלב סינונים:
```
חיפוש: "שוקולד"
קטגוריה: "ממתקים וחטיפים"
מחיר: 5-15 ₪
מיון: "מחיר נמוך לגבוה"
```

### 3. השתמש בקטגוריות:
במקום לחפש "חלב קוטג'" - פשוט סנן:
- קטגוריה: "חלב ומוצרי חלב"
- מיון: "מחיר נמוך לגבוה"

## 🛒 דוגמאות שימוש

### מקרה 1: מחפש את החלב הזול ביותר
```
1. חפש: "חלב טרי"
2. סנן: קטגוריה = "חלב ומוצרי חלב"
3. מיין: "מחיר: נמוך לגבוה"
4. תוצאה: החלב הזול ביותר ראשון!
```

### מקרה 2: השוואת מחירים בין סניפים
```
1. חפש מוצר ספציפי (למשל: "קוקה קולה 1.5")
2. לחץ "פרטים"
3. תראה רשימה של כל הסניפים והמחירים שלהם
```

### מקרה 3: מחפש מוצרים זולים לילדים
```
1. קטגוריה: "תינוקות"
2. מחיר מקסימלי: 50
3. מיין: "מחיר: נמוך לגבוה"
```

### מקרה 4: מחפש חטיפים ב-10 ש"ח
```
1. חיפוש: "חטיף"
2. מחיר: 5-15
3. קטגוריה: "ממתקים וחטיפים"
```

## ❓ שאלות נפוצות

### למה יש מוצר במחיר 0.01 ₪?
זה כנראה מוצר במבצע או טעות בנתונים. תמיד כדאי לבדוק את כל המחירים ב"פרטים".

### למה אותו מוצר מופיע פעמיים?
ייתכן שזה אותו מוצר בגודל שונה או באריזה שונה. בדוק את הברקוד לוודא.

### איך אני יודע מאיזה סניף המחיר?
תראה בעמודת "חנויות" את שם הסניף. לפרטים מלאים לחץ על "פרטים".

### המערכת מעודכנת?
כל מחיר מציג "עודכן" - זה אומר מתי הנתונים נלקחו מהמערכת של החנות.
