# .agent Directory

תיקייה זו מכילה הנחיות ותיעוד עבור AI agents שעובדים על הפרויקט.

## קבצים

### `rules.md` ⚠️ **חובה לקרוא!**
כללים והנחיות לעבודה על הפרויקט:
- ארגון קבצים
- כללי קוד
- תהליך עבודה
- בדיקות חובה
- שגיאות נפוצות

### `session-summary.md`
סיכום מהיר של המצב הנוכחי:
- מה נעשה לאחרונה
- סטטוס מסד הנתונים
- כלים זמינים
- TODO list

## שימוש

כל AI agent שמתחיל לעבוד על הפרויקט צריך:
1. לקרוא את `rules.md` **קודם כל**
2. לעיין ב-`session-summary.md` להבנת המצב
3. לעקוב אחר ההנחיות בזמן העבודה
4. לעדכן את `session-summary.md` בסוף כל שיחה
