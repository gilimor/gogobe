## דברים שכדאי לדעת

### הבדלי רמות:
- **Mid-Grade** = גרסה זולה יותר
- **Standard** = גרסה רגילה
- **Delicate** = גרסה עדינה יותר
- **TC (Tungsten Carbide)** = גרסה מחוזקת
- **SklarLite™** = גרסה קלה (Premium)

### הבדלי צורה:
- **Straight** = ישר
- **Curved** = מעוקל
- **Angled** = בזוית
- **Single End** = קצה אחד
- **Double End** = שני קצוות

### הבדלי גודל:
- **Child/Pediatric** = לילדים
- **Adult** = למבוגרים
- **Small/Medium/Large** = קטן/בינוני/גדול
