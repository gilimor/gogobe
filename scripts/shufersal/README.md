# Shufersal Scripts

סקריפטים לייבוא נתונים מרשת שופרסל.

## קבצים

### `import.bat`
יבוא קבצים בודדים או מספר קבצים מסניפי שופרסל.

**שימוש:**
```bash
# הרץ מתיקיית הפרויקט
scripts\shufersal\import.bat
```

### `import-all.bat`
יבוא אוטומטי של כל קבצי שופרסל שהורדו.

**שימוש:**
```bash
scripts\shufersal\import-all.bat
```
